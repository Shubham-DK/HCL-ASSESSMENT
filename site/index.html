Thought process behind Implementation

  • Since a lot of frequent update is happening,hence used Map to store the responses, because it performs better than other data structures
  • Created Separate Modules for Separation of Concerns and they are listed below:
    • Handle Response Module - Only handles the responses and forward it to Data Manager Module
    • Data Manager Module - This Modules sort and store the responses and forward it to Grid Module
    • Grid Module - This Modules create the table and update it based on latest data
  • NOTE : Storing only last 30secs of data for drawing sparkline
